Backend Test


1. View entire table in database:

https://backend-testing.herokuapp.com/db

2. View only Likes (includes ID) from table:

https://backend-testing.herokuapp.com/db/like

3. View only Views (includes ID) from table:

https://backend-testing.herokuapp.com/db/views

4. Query a number to the database:

This will return records that has the queried number in any of the fields (ID, Money, Like, Views, Share)

Enter a number after the "=" sign in the URL

https://backend-testing.herokuapp.com/db/values?values=

5. Query a field name to the database:

This will return all records with only the queried field

Enter a field name (id, date, money, like, views, share) after the "=" sign in the URL

Note: To query "like" field, enter "data.like" after "=" sign!

https://backend-testing.herokuapp.com/db/field?field=
